# Nigel's Kitchen Table — Brand Assets

These files are the approved logo assets referenced by `docs/brand-master.md`.

## Files

- `nkt-logo-horizontal.svg` — primary logo for desktop headers, emails and wide layouts.
- `nkt-logo-compact.svg` — compact wordmark for narrower layouts.
- `nkt-monogram.svg` — small-format mark for favicons, profile images and square applications.

## Mailchimp

Mailchimp email image blocks work best with PNG versions of the logos. Use the horizontal logo as the primary Mailchimp logo and the monogram only for small square placements.

Recommended export sizes:

- Horizontal logo: 1040 × 264 px, transparent PNG.
- Compact logo: 600 × 200 px, transparent PNG.
- Monogram: 512 × 512 px, PNG.

Do not recolour, stretch or combine these marks with a prominent The Gourmet Larder logo.
