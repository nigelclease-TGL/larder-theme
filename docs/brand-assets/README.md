# Nigel's Kitchen Table — Brand Assets

These files are the approved visual assets referenced by `docs/brand-master.md`.

## Files

- `nkt-logo-horizontal.svg` — primary logo for desktop headers, emails and wide layouts.
- `nkt-logo-compact.svg` — compact wordmark for narrower layouts.
- `nkt-monogram.svg` — small-format mark for favicons, profile images and square applications.
- `nkt-brand-moodboard.svg` — approved visual moodboard showing the intended food photography, palette, typography, editorial spacing and kitchen-table atmosphere.

## Moodboard use

Use the moodboard as a visual reference when creating social covers, campaign graphics, Pinterest assets, newsletter designs and future website sections. It supports the written rules in `docs/brand-master.md`; it does not replace the approved logo files or canonical colour values.

The moodboard should guide:

- warm natural-light food photography,
- restrained olive, ivory, linen and copper tones,
- calm editorial layouts with generous spacing,
- desserts, bakes, bread, seasonal ingredients and practical kitchen details,
- food-led presentation with Nigel's portrait remaining secondary.

## Mailchimp

Mailchimp email image blocks work best with PNG versions of the logos. Use the horizontal logo as the primary Mailchimp logo and the monogram only for small square placements.

Recommended export sizes:

- Horizontal logo: 1040 × 264 px, transparent PNG.
- Compact logo: 600 × 200 px, transparent PNG.
- Monogram: 512 × 512 px, PNG.

Do not recolour, stretch or combine these marks with a prominent The Gourmet Larder logo.
