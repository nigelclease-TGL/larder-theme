## Image rules

- Keep the existing presentation layout.
- Replace photographs only when Nigel explicitly requests it.
- Use genuine Nigel's Kitchen Table recipe, website or approved social-media photographs.
- Do not use AI-generated food photographs.
- Preserve natural image proportions and avoid distorted or awkward crops.

- Update verified audience figures before external use.
- Do not invent website traffic, demographic percentages, campaign results or testimonials.
- Use `Rates available upon request` until a rate card is approved.
- Keep the brand aligned with `../brand-master.md` and `../media-kit.md`.
