# Nigel's Kitchen Table — Editorial Workflow

Version 1.9.0 adds a read-only editorial audit under **Tools → Recipe Content Audit**. It does not rewrite posts, alter permalinks or publish changes.

## Recipe essentials

A recipe is considered publish-ready when it has:

- a featured image,
- a useful category rather than only Uncategorized,
- a concise editorial excerpt,
- a WP Recipe Maker card.

The audit also recommends descriptive alt text and enough supporting copy to answer likely reader questions.

## Kitchen Note essentials

A Kitchen Note is detected through the `kitchen-notes` or `baking-guides` category. It should have:

- a featured image,
- descriptive alt text,
- a useful excerpt,
- a useful category,
- enough editorial depth to be genuinely helpful.

Kitchen Notes keep visible publication dates. Evergreen recipe posts do not display publication dates on the public website.

## Using the audit

1. Open **Tools → Recipe Content Audit**.
2. Start with **Missing essentials**.
3. Open each post in the editor and address the checklist in the **Kitchen Table readiness** panel.
4. Then review **Needs attention** for alt text and supporting-copy recommendations.
5. Export a CSV when planning a larger content-improvement sprint.

The Posts screen also includes **Kitchen Table type** and **Content readiness** columns.
